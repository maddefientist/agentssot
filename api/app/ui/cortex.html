How the Neural Net Works

5-Layer Architecture
Layer 1 — Memory: Every AI conversation generates knowledge. AgentSSOT captures facts, decisions, and patterns as knowledge items with vector embeddings.

Layer 2 — Feedback: When recalled knowledge is useful, agents send useful/noted/wrong signals. Useful concepts gain confidence; wrong ones get tagged contested.

Layer 3 — Skills: Action patterns crystallize into skills with trigger (when), action (do what), and success_hint (verify). Shown as green nodes.

Layer 4 — Personalization: Each agent/device builds a profile of strengths from its recall patterns. Recall results are boosted toward each agent's expertise areas.

Layer 5 — Collective Intelligence: When 3+ agents independently confirm a concept, it reaches consensus (gold glow). Cross-agent confirmation elevates truth.

Concept Lifecycle
Knowledge items cluster by embedding similarity. Ollama synthesizes concepts. Reconciler merges or creates. Confidence grows with evidence (+0.1 per match) and agent confirmation (+0.05 per new agent). Stale concepts decay (-0.02/cycle after 90 days). Dormant at floor (0.15). Resurrectable on re-recall.

Visual Encoding
Color = concept type. Size = evidence count. Opacity = confidence. Ring = scope (inner=global, mid=project, outer=device). Gold glow = consensus.

Onboarding & Troubleshooting

Enroll a new device
1. From an admin session, create an enrollment token
2. On the new device, call POST /auto-enroll
3. Save the returned agent config to ~/.claude/agentssot/local/agent.json

Verify connectivity
curl -s http://YOUR_HOST:8088/health | python3 -m json.tool
Test recall
curl -s -X POST http://YOUR_HOST:8088/recall \ -H "X-API-Key: YOUR_KEY" \ -H "Content-Type: application/json" \ -d '{"query_text":"test","namespace":"claude-shared"}' | python3 -m json.tool
Trigger synthesis manually
curl -s -X POST http://YOUR_HOST:8088/synthesis/run \ -H "X-API-Key: ADMIN_KEY" \ -d '{"namespace":"claude-shared"}' | python3 -m json.tool

Troubleshooting
- embedding_available: false — Check Ollama is running
- llm_available: false — Verify Ollama and model pulled
- No concepts appearing — Run synthesis manually then wait 10s
- Recall returns empty — Check namespace and knowledge items
